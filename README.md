# Comp Sci Student Simulator
Comp Sci Student Simulator is a fully functional video game developed as a final project for CSCI 198. Built collaboratively by a three-person team, the game bridges conceptual game design with low-level technical implementation using C++ and OpenGL.

Key Technical Achievements:
- Object-Oriented Architecture: Designed a scalable, object-oriented project structure to handle game states, rendering, and entity management.
- Legacy Framework Navigation: Successfully engineered the game using the older GLUT library, overcoming significant hurdles with outdated dependencies and file structures to deliver a stable, compiled build.
- Team Collaboration: Managed version control, code integration, and the full software development lifecycle in a fast-paced team environment.

Built With:
- C++
- GLUT (OpenGL Utility Toolkit)
- OpenGL
